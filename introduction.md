# LinkedIn Learning - React:Creating and Hosting a Full-Stack Site

Shaun Wassell is the instructor for this course


## Introduction 

### Build your own fullstack website using React
---
Imagine building a React Full-Stack application to learn React and build a real world project to show potential employers. This could make a good impression in a job interview.

This course introduces the MERN stack which includes:
  - MongoDB
  - Express
  - React
  - Node

At the end you will have a hosted full-stack app you can show off to employers and friends.

### What you should know
---
This course assumes you have the following knowledge:
 - basics of JavaScript
 - basics of React
 - basic knowledge of networking such as terms such as client & server
